# LangChain Basics

LangChain 기초 학습을 위한 프로젝트입니다.

## 프로젝트 구조

- `1. chat.ipynb` - ChatOllama를 사용한 기본 채팅 예제
- `2. prompt.ipynb` - PromptTemplate 사용 예제
- `3. output parser.ipynb` - Output Parser 사용 예제
- `4. runnable parser.ipynb` - Runnable과 Parser 조합 예제

## 사용 기술

- LangChain
- Ollama (로컬 LLM)
- Python 3.11

## 설치 방법

```bash
pip install langchain-ollama langchain-core
```

## 사용 방법

각 노트북 파일을 Jupyter에서 열어 실행.
